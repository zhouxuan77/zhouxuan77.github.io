【Git】Git 同时连接 Github 和 Gitee

2021年08月16日 368 字 大概 1 分钟

究其原因，Github 的访问速度太拉垮了，迫于无奈。先用 Gitee push，再转存到 Github 上

移除 Git 的全局属性

  1. 开始菜单 - Git - Git bash （打开 Git 命令行）
  2. 移除全局配置账户
  3. 移除全局配置邮箱
  4. 移除全局密码
  • 其他查看命令

生成用户的 SSH KEY

  1. 开始菜单 - Git - Git bash （打开 Git 命令行）
  2. 生成对应 Github 的 SSH KEY
  3. 生成对应 Gitee 的 SSH KEY
  4. 打开路径：C:\Users\用户名\.ssh
  5. 将公钥添加到 Github 和 Gitee 上 太懒了，不高兴截图

配置 config 文件

  1. 在 .ssh 目录下创建一个 config 文件 （不是TXT文本）
  2. 输入参数模板
  3. 参数示例
  4. 设置用户名和邮箱
  5. 测试用户的 SSH 密钥是否生效
    • 测试 Github
    • 测试 Gitee
    • 弹出该文本即密钥生效
