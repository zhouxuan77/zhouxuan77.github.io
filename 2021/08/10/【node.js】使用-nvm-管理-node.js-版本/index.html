【node.js】使用 nvm 管理 node.js 版本

2021年08月10日 321 字 大概 1 分钟

记录一下如何使用 nvm 管理 node.js 版本

node.js 安装

  1. 下载 nvm
    • 下载地址：https://github.com/coreybutler/nvm-windows/releases
    • 网盘地址：暂无
  2. 安装 nvm
    • nvm-noinstall.zip：绿色免安装版本，使用前需要配置
    • nvm-setup.zip：安装包，使用前无需配置
    • Source code(zip)：zip 压缩的源码
    • Source code(tar.gz)：tar.gz 的源码
    • 安装步骤一直 next 即可
    • 如需要更改路径，路径不得出现中文和空格
  3. 配置 nvm
    • 在 nvm 安装路径下找到 setting.txt [^注1]
      [^注1]：root：nvm 路径；path：nodejs 路径

  4. 下载对应版本的 node.js

    • 安装对应版本的 nodejs

node.js 环境配置

  • 在node和npm不能使用时，可以手动添加node的环境变量

  1. 打开系统环境变量

  2. 在用户变量处，添加 NVM_HOME 和 NVM_SYMLINK

  3. 在系统变量处，添加 %NVM_HOME% 和 %NVM_SYMLINK%
    ![04.PNG]](https://i.loli.net/2021/08/11/cXplmON6GPKakfZ.png)

  4. 启动 CMD ，输入 node -v 和 npm-v，显示版本号即可
