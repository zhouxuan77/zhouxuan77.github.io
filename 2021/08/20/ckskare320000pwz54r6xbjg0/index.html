【hexo】创建属于自己的博客

2021年08月20日 276 字 大概 1 分钟

突然间，心血来潮。然后就想着创建一个自己的博客，顺带着可以记录一下自己的学习历程和参与过的项目。

前期准备

  1. 安装 nodejs （推荐版本为：12.14.0）
    • nodejs 安装说明
  2. 配置 Git
    • Git 配置说明

安装配置 Hexo

  1. 配置好 nodejs 开发环境后，打开 CMD 输入
  2. 初始化并建站
  3. 打开 _config.yml，并修改对应的参数
    • hexo 配置

安装配置对应的主题

  1. 打开 CMD，进入对应的路径
  2. 安装主题
  3. 安装 WordCount
  4. 启用 Nexmoe 主题
    • 在 config.yml 中，修改 theme 的值为 nexmoe
  5. 配置 Nexmoe 主题
    • 根目录下修改 _config.nexmoe.yml
  6. 具体配置参数
    • Nexmoe 配置

Hexo 常用指令

  1. 新建一篇文章
  2. 生成静态文件
  3. 启动服务器 （需要先进行安装：npm install hexo-server）
  4. 部署网站
