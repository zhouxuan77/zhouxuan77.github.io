【hexo】加密博客文章

2021年08月20日 304 字 大概 1 分钟

有的时候需要写一些私密的博客, 所以通过密码验证的方式可以让人不能随意浏览

注意事项

  1. 博客需要 https ，否则无法正常解密并访问

安装

npm install --save hexo-blog-encrypt

# 或使用 yarn 安装
yarn add hexo-blog-encrypt

快速使用

  1. 将 password 字段添加到文章信息头
  2. 修改 文章信息头 或修改 _config.yml
    • 文章信息头
    • _config.yml
